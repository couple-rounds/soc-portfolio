# Security Operations Portfolio

Hands-on security operations portfolio focused on detection engineering, threat hunting, and incident response.

**Target roles:** SOC Analyst (Tier 1 and Tier 2)

## Focus areas

| Area | Coverage |
| --- | --- |
| Splunk | SPL queries, dashboards, and alert investigations |
| Microsoft Sentinel | KQL analytics rules, Microsoft 365 log onboarding, and incident response runbooks |
| Detection engineering | Sigma rules mapped to MITRE ATT&CK |
| Threat hunting | Hypothesis-driven investigations and reports |
| Incident response | Detection, containment, recovery, and documentation |
| Security foundations | Linux, Windows, networking, cyber threat intelligence, the Cyber Kill Chain, and the Diamond Model |

Detection rules include relevant [MITRE ATT&CK](https://attack.mitre.org/) technique mappings.

## Documentation standards

- Explain the objective, methodology, findings, and outcome of each project
- Include reproducible queries, rules, and supporting evidence where appropriate
- Map detections to MITRE ATT&CK techniques
- Record limitations and potential improvements
